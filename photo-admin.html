Photo Manager

Add, swap, or remove photos across the Writing Center website. Changes generate HTML code you paste into the page.

How to Update Photos

1
Click Upload Photo on any slot to choose an image from your computer, or click Link URL to paste in a web address for an existing photo.
2
Edit the caption tag (e.g. "One-on-One") and description for each photo. These appear on the website under each image.
3
Click Generate Code — it creates the HTML block you copy and paste into index.html to replace the photo section.
4
For hosted images (recommended): Upload photos to your server or a service like Cloudinary / Google Drive, copy the public URL, and use "Link URL" to paste it in.
